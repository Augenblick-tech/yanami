# 追番service

简化追番流程，自动化订阅新番更新

> 服务启动后会通过bangumi获取当前正在更新的番剧列表  
> 在tmdb上查出来对应番的季度信息和日语原名、简中翻译、繁中翻译  
> 通过配置的RSS全站订阅链接和搜索链接去轮询更新  
> 将从RSS获取到的种子资源和预设的番剧规则进行匹配，命中规则便推送qbit下载  
> 通过识别番剧的剧集和记录下载历史，来判断番剧是否更新完结，是则停止监听更新
